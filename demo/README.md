# Тестовые файлы

Эта папка содержит файлы для проверки работы скрипта `SPOT_WHITE.jsx`:

- **test_image.png** — тестовое растровое изображение на прозрачном фоне. Вы можете открыть его в Photoshop или пересохранить в PDF.
- **test_document.pdf** — готовый тестовый PDF-файл (размер 1000x1000 пикселей) с прозрачным фоном и геометрическими фигурами. Идеально подходит для проверки скрипта, так как скрипт ожидает на входе именно PDF-файл.

**Как проверить:**
1. Запустите скрипт `SPOT_WHITE.jsx` в Photoshop.
2. Выберите файл `test_document.pdf` из этой папки.
3. Проверьте результат (TIFF файл с созданным плашечным каналом `Spot_White`).
