# Demo Files

This folder contains sample files to test the `SPOT_WHITE.jsx` script:

- **test_image.png** — A sample raster image with a transparent background. You can open it in Photoshop or use it to create your own PDFs.
- **test_document.pdf** — A ready-to-use PDF file (1000x1000 pixels) with a transparent background and simple geometric shapes. This file is ideal for testing since the script explicitly prompts for a PDF.

**How to test:**
1. Run the `SPOT_WHITE.jsx` script in Photoshop.
2. Select the `test_document.pdf` file from this folder.
3. Verify the generated output (a TIFF file with the newly created `Spot_White` channel).
